# Carnis

## Description

A simple racing arcade simulator with [Box2D](https://box2d.org/) physics, featuring an unsettling theme of escaping the guts of an Forsaken Elder God.

[Original Repository](https://github.com/JiaHaoZhao05/PHY2)

## Key Features

* Player can shoot two different obstacles to disrupt enemies
* Each enemy has different behavior and reactions
* Variety items that drastically changes gameplay
* Slightly uncomfortable theme

## Controls

* W/Up arrow -> Throttle 
* D/Right arrow -> Turn right
* A/Left arrow -> Turn left
* S/Down arrow -> Reverse
* Shift -> Brake
* Enter -> Start game
* Key F1 -> Debug

 	\*(While debugging you can click and drag the ball)
